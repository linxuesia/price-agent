# 门窗报价助手 — 客户部署指南

## 环境要求

- **操作系统**：macOS / Windows / Linux
- **Python**：3.8 或以上版本
- **网络**：需要能访问互联网（调用智谱 AI 接口）
- **磁盘空间**：约 500MB（含 Chromium 浏览器）

## 第一步：安装 Python

### macOS
```bash
brew install python@3.11
```
或从 https://www.python.org/downloads/ 下载安装包。

**验证安装**：
```bash
python3 --version
# 应输出 Python 3.8 或更高版本
```

### Windows
1. 打开 https://www.python.org/downloads/
2. 下载最新版安装包，**安装时勾选「Add Python to PATH」**
3. 验证：打开命令提示符，输入 `python --version`

## 第二步：复制项目文件

将以下文件复制到客户电脑的任意目录（如桌面上的 `报价助手` 文件夹）：

```
报价助手/
├── app.py           ← 主程序
├── chat.html        ← 网页界面
├── calc.py          ← 计算模块
├── pricing.json     ← 定价数据
├── requirements.txt ← Python 依赖列表
├── setup.sh         ← macOS 安装脚本
├── setup.bat        ← Windows 安装脚本
├── 启动.command      ← macOS 启动器
└── run.bat         ← Windows 启动器
```

> 不需要复制 `pages/`、`*.pdf`、`results*.json` 等测试文件。

## 第三步：安装依赖

### macOS
双击 `setup.sh`，或在终端中运行：
```bash
cd ~/Desktop/报价助手
bash setup.sh
```

### Windows
双击 `setup.bat`，或在命令提示符中运行：
```cmd
cd C:\Users\客户\Desktop\报价助手
setup.bat
```

## 第四步：启动服务

### macOS
双击 `启动.command`，浏览器会自动打开。

或手动：
```bash
cd ~/Desktop/报价助手
source venv/bin/activate
python app.py
```

### Windows
双击 `run.bat`，浏览器会自动打开。

或手动：
```cmd
cd C:\Users\客户\Desktop\报价助手
venv\Scripts\activate
python app.py
```
浏览器打开 http://localhost:8080

## 使用说明

1. **上传图纸**：点击上传区域或拖拽 PDF 文件
2. **等待识别**：AI 自动逐页识别门窗尺寸、型材、玻璃规格
3. **查看报价单**：生成后自动下载，也可点击图片放大查看
4. **配置定价**：点击右侧「定价配置」面板，修改单价后保存
5. **API Key**：点击标题栏齿轮图标，可填入自己的智谱 API Key（留空使用内置 Key）

## 停止服务

在终端中按 `Ctrl + C`，或直接关闭终端窗口。

## 常见问题

**Q: 启动后浏览器显示"无法连接"**
- 确认终端没有报错
- 检查是否在正确的目录下运行
- 确认 `python app.py` 输出中包含 `localhost:8080`

**Q: 上传 PDF 后一直转圈**
- 检查网络连接（需要访问智谱 API）
- 可能是 API Key 失效，点击齿轮图标换一个 Key

**Q: 生成的报价单价格不对**
- 点击右侧「定价配置」面板，检查并修改单价，保存后重新上传

**Q: 每次都要手动启动吗？**
- 是的，这是本地服务。可以设置开机自启（Windows 任务计划程序 / macOS 登录项）

核心能力

AI 视觉识别 自动报价计算 PDF 解析 实时进度展示 飞书 Bot 集成 一键部署

  • 调用智谱 GLM-4V多模态大模型，精准识别门窗尺寸、型材、玻璃规格
  • 自动计算面积、开启扇、玻璃加厚、纱窗、密码锁等费用
  • 美观的聊天式界面，SSE 实时推送识别进度
  • 支持 macOS / Windows 一键安装部署

快速开始

环境要求: Python 3.8+ | 网络连接（访问智谱 AI API）

git clone https://github.com/linxuesia/price-agent.git
cd price-agent
pip install -r requirements.txt
cp .env.example .env          # 编辑 .env 填入 API Key
python app.py                  # 浏览器打开 http://localhost:8080

仓库包含脱敏示例图纸 sample.pdf，上传即可测试。
